# Lenskart UI/UX Concept Prototype

> **Built for the Code Canvas**
> This project was developed as an intensive design and development challenge, focusing on high-impact UI/UX and real-time 3D integration.Our goal was to prove that e-commerce experiences—specifically for lenskart — can be both beautiful and highly functional.


# WHAT ACTUALLY PROJECT IS

A clean, user-centric e-commerce experience designed for first-time eyewear buyers. This prototype focuses on product discovery, interactive , and a zero-friction checkout flow.
